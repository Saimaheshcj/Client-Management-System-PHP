# Client Management System

A web-based application built using PHP and MySQL to manage client data securely.

## Features
- Login Authentication
- Add / Edit / Delete Clients
- Dashboard
- Database Integration

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
